VAS Label Generator

Tạo tem COO (Country of Origin) từ Master Label Data + EAN List — chạy hoàn toàn trong trình duyệt, dữ liệu không gửi lên server nào.

1. Master Label Data khoá & tái sử dụng

File Excel master (nhiều sheet OK — bạn chọn đúng sheet ở popup tiếp theo). Lưu trong trình duyệt của bạn, không upload lên server nào.

Chưa có master data nào được khoá cho mục này.

2. EAN / Filter List

File cần có tối thiểu 2 trong 3 cột: EAN, SKU, CO (Country of Origin). Upload lại mỗi lần tạo tem.

Chưa upload file EAN/filter.

3. Font chữ khoá & tái sử dụng

Mặc định dùng Noto Sans SC (mã nguồn mở). Muốn dùng đúng SimHei/SimSun thật, upload file .ttf/.ttc — font sẽ được khoá lại trong trình duyệt, dùng cho mọi lần tạo tem sau này.

Đang dùng font mặc định: Noto Sans SC.

4. Tuỳ chọn tem & cỡ chữ

5. Tạo tem

Chọn sheet chính xác của Master Label Data

File có nhiều sheet. Hãy chọn sheet chứa dữ liệu tem cần dùng.

Font mặc định: Noto Sans SC (mã nguồn mở, OFL) — có thể thay bằng SimHei/SimSun thật trong fonts/. Xem README.md.
